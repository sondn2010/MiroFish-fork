## ⚡ Tổng Quan

**MiroFish** là một công cụ dự đoán AI thế hệ mới được cung cấp bởi công nghệ đa tác nhân. Bằng cách trích xuất thông tin hạt giống từ thế giới thực (như tin tức tức thời, dự thảo chính sách hoặc tín hiệu tài chính), nó tự động xây dựng một thế giới kỹ thuật số song song có độ trung thực cao. Trong không gian này, hàng ngàn tác nhân thông minh với nhân cách độc lập, bộ nhớ dài hạn và logic hành vi tương tác tự do và trải qua sự tiến hóa xã hội. Bạn có thể động thêm các biến số từ góc nhìn "Của Thiên Chúa" để suy ra chính xác các quỹ đạo tương lai — **tập dợt tương lai trong một hộp cát kỹ thuật số, và giành chiến thắng các quyết định sau vô số mô phỏng**.

> Bạn chỉ cần: Tải lên các tài liệu hạt giống (báo cáo phân tích dữ liệu hoặc các câu chuyện tiểu thuyết thú vị) và mô tả yêu cầu dự đoán của bạn bằng ngôn ngữ tự nhiên</br>
> MiroFish sẽ trả về: Một báo cáo dự đoán chi tiết và một thế giới kỹ thuật số song song có độ trung thực cao và có khả năng tương tác sâu

### Tầm Nhìn Của Chúng Tôi

MiroFish được dâng hiến để tạo ra một tấm gương trí thông minh bầy đàn ánh xạ thực tế. Bằng cách nắm bắt sự nổi lên tập thể được kích hoạt bởi các tương tác cá nhân, chúng tôi vượt qua những hạn chế của dự đoán truyền thống:

- **Ở Cấp Độ Vĩ Mô**: Chúng tôi là một phòng thí nghiệm tập dợt cho các nhà hoạch định chính sách, cho phép các chính sách và quan hệ công chúng được kiểm tra không rủi ro
- **Ở Cấp Độ Vi Mô**: Chúng tôi là một hộp cát sáng tạo cho người dùng cá nhân — cho dù suy ra những kết thúc tiểu thuyết hay khám phá các kịch bản trí tưởng tượng, mọi thứ đều có thể vui vẻ, đáng chơi và dễ tiếp cận

Từ những dự đoán nghiêm túc đến mô phỏng mang tính chơi đùa, chúng tôi cho phép mọi "nếu như" nhìn thấy kết quả của nó, làm cho việc dự đoán mọi thứ trở thành có thể.

## 🔄 Quy Trình Làm Việc

1. **Xây Dựng Biểu Đồ**: Trích xuất hạt giống & Tiêm nhớ cá nhân/tập thể & Xây dựng GraphRAG
2. **Cài Đặt Môi Trường**: Trích xuất mối quan hệ thực thể & Tạo Nhân Vật & Tiêm cấu hình Agent
3. **Mô Phỏng**: Mô phỏng song song hai nền tảng & Tự động phân tích yêu cầu dự đoán & Cập nhật bộ nhớ thời gian động
4. **Tạo Báo Cáo**: ReportAgent với bộ công cụ phong phú để tương tác sâu với môi trường sau mô phỏng
5. **Tương Tác Sâu**: Trò chuyện với bất kỳ tác nhân nào trong thế giới được mô phỏng & Tương tác với ReportAgent

## 🚀 Bắt Đầu Nhanh Chóng

### Tùy Chọn 1: Triển Khai Mã Nguồn (Được Khuyến Nghị)

#### Các Yêu Cầu Tiên Quyết

| Công Cụ | Phiên Bản | Mô Tả | Kiểm Tra Cài Đặt |
|------|---------|-------------|-------------------|
| **Node.js** | 18+ | Môi trường chạy tiền end, bao gồm npm | `node -v` |
| **Python** | ≥3.11, ≤3.12 | Môi trường chạy backend | `python --version` |
| **uv** | Mới nhất | Trình quản lý gói Python | `uv --version` |

#### 1. Cấu Hình Các Biến Môi Trường

```bash
# Sao chép tệp cấu hình ví dụ
cp .env.example .env

# Chỉnh sửa tệp .env và điền các khóa API cần thiết
```

**Biến Môi Trường Bắt Buộc:**

```env
# Cấu Hình API LLM (hỗ trợ bất kỳ API LLM nào với định dạng OpenAI SDK)
# Được khuyến nghị: Mô hình Alibaba Qwen-plus qua Nền tảng Bailian: https://bailian.console.aliyun.com/
# Tiêu thụ cao, hãy thử mô phỏng với ít hơn 40 vòng trước tiên
LLM_API_KEY=your_api_key
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL_NAME=qwen-plus

# Cấu Hình Zep Cloud
# Hạn mức miễn phí hàng tháng là đủ cho việc sử dụng đơn giản: https://app.getzep.com/
ZEP_API_KEY=your_zep_api_key
```

#### 2. Cài Đặt Các Phụ Thuộc

```bash
# Cài đặt một lần tất cả các phụ thuộc (gốc + tiền end + backend)
npm run setup:all
```

Hoặc cài đặt từng bước:

```bash
# Cài đặt các phụ thuộc Node (gốc + tiền end)
npm run setup

# Cài đặt các phụ thuộc Python (backend, tự động tạo môi trường ảo)
npm run setup:backend
```

#### 3. Bắt Đầu Dịch Vụ

```bash
# Bắt đầu cả tiền end và backend (chạy từ gốc dự án)
npm run dev
```

**URL Dịch Vụ:**
- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:5001`

**Bắt Đầu Riêng Lẻ:**

```bash
npm run backend   # Chỉ bắt đầu backend
npm run frontend  # Chỉ bắt đầu frontend
```

### Tùy Chọn 2: Triển Khai Docker

```bash
# 1. Cấu hình các biến môi trường (giống như triển khai nguồn)
cp .env.example .env

# 2. Kéo hình ảnh và bắt đầu
docker compose up -d
```

Đọc `.env` từ thư mục gốc theo mặc định, ánh xạ các cổng `3000 (tiền end) / 5001 (backend)`

> Địa chỉ gương để kéo nhanh hơn được cung cấp dưới dạng chú thích trong `docker-compose.yml`, thay thế nếu cần.
